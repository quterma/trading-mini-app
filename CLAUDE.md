# Инструкции для Claude

## Стиль коммуникации
- Все ответы на русском языке
- Максимум 50 слов в сообщениях
- Без примеров кода в текстовых ответах
- Никаких эмодзи без явного запроса

## Git коммиты
- Сообщения до 30 слов
- Без подписи соавторства (Co-Authored-By)
- Без результатов проверок
- Только описание изменений

## Архитектура проекта
- Минимальная FSD
- Без стейт-менеджеров
- Все фичи через публичные index.ts
- Импорты через @/ алиасы
- Мобильный-first дизайн

## Технологии
- React 19 + TypeScript
- Vite
- Tailwind CSS
- Recharts для графиков
- clsx для className

## Правила разработки
- Prettier без точек с запятой (semi: false)
- ESLint + Prettier
- Без глубоких импортов
- Только именованные экспорты через index.ts
